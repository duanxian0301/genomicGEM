# Results Folder

Use this folder for shareable analysis outputs that are small enough to version safely.

Recommended contents:

- summary tables for LDSC, EFA, CFA, and factor GWAS
- manuscript-ready supplementary tables
- figure source tables
- small metadata files describing export columns and sample definitions

Avoid committing:

- very large chunk-level GWAS outputs
- compressed intermediate files larger than necessary
- temporary scratch files

If needed, create subfolders such as:

- `results/tables/`
- `results/figures/`
- `results/metadata/`
